# To-Do List App

A simple to-do list application built with HTML, CSS and JavaScript.  
Users can add tasks, mark them as completed, and delete them from the list.

## Features

- Add new tasks using a text input
- Display all tasks in a dynamic list
- Mark tasks as completed (line-through style)
- Delete individual tasks
- Basic validation to ignore empty input

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## How to Run

1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. Type a task in the input field and click **Add**.

## Project Structure

- `index.html` – main page
- `style.css` – styling for the app
- `script.js` – JavaScript logic for managing tasks
